✨ Características

Un invernadero inteligente

🌡️

Monitoreo en tiempo real

Temperatura, humedad, luminosidad y pH registrados cada 30 segundos con alertas automáticas ante anomalías.

💧

Riego automatizado

El sistema activa el riego por goteo automáticamente cuando la humedad del suelo baja del umbral configurado.

📱

Acceso desde cualquier lugar

Dashboard web responsive: consultá el estado del invernadero desde tu celular, tablet o computadora.

📋

Historial completo

Todas las lecturas quedan registradas en base de datos para análisis histórico y seguimiento de tendencias.

🌱

Gestión de cultivos

Seguimiento del ciclo de vida de cada cultivo con alertas de cosecha y recomendaciones de cuidado.

🔔

Sistema de alertas

Notificaciones en tiempo real ante condiciones críticas: temperatura fuera de rango, sensores sin señal y más.
