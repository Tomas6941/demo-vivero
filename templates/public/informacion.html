📖 El proyecto

¿Qué es el Vivero Biotecnológico?

El Vivero Biotecnológico de la Escuela ProA Capital es un proyecto interdisciplinario que integra tecnología IoT, automatización y ciencias naturales para crear un invernadero inteligente gestionado por los propios estudiantes.

El sistema permite monitorear en tiempo real las condiciones ambientales del invernadero —temperatura, humedad, luminosidad y pH del suelo— y actuar sobre ellas automáticamente mediante actuadores controlados por software.

Objetivos del proyecto

  • 🎯 Desarrollar un sistema de monitoreo ambiental para el invernadero escolar.
  • 🎯 Automatizar el riego, la ventilación y la iluminación según parámetros configurables.
  • 🎯 Visualizar los datos en un dashboard web accesible desde cualquier dispositivo.
  • 🎯 Registrar el historial completo de lecturas para análisis y aprendizaje.
  • 🎯 Integrar conocimientos de hardware, software, biología y gestión de proyectos.
  • 🎯 Generar documentación técnica completa como legado para futuros estudiantes.

¿Cómo funciona el sistema?

El sistema opera en cuatro capas interconectadas:

1
Capa física: Sensores DHT22, BH1750 y sensor de pH instalados en distintos sectores del invernadero capturan datos del entorno cada 30 segundos.
2
Capa de procesamiento: Una Raspberry Pi 4 actúa como controlador central, recopila las lecturas via GPIO/I²C y ejecuta la lógica de automatización.
3
Capa de servidor: Flask (Python) expone una API REST que almacena los datos en SQLite y sirve la interfaz web.
4
Capa de presentación: El dashboard web muestra los datos en tiempo real con gráficos, alertas y controles de los actuadores.
🔌 Tecnologías · Hardware

Hardware utilizado

🖥️

Raspberry Pi 4

Controlador central del sistema. Gestiona sensores, actuadores y el servidor web desde un único dispositivo de bajo consumo.

⚡

Arduino Uno

Microcontrolador auxiliar para la lectura de sensores analógicos y control de relés de los actuadores físicos.

💧

Bomba de agua

Bomba peristáltica de 12V controlada por relé para el sistema de riego por goteo automatizado.

💨

Ventiladores DC

Ventiladores de 12V controlados por PWM para regulación de temperatura y circulación de aire.

🌡️ Tecnologías · Sensores

Sensores instalados

🌡️

DHT22

Sensor digital de temperatura (−40 a 80 °C) y humedad relativa (0–100%). Alta precisión y comunicación de un solo cable.

Temperatura · Humedad
☀️

BH1750

Sensor de luz ambiental digital con interfaz I²C. Rango de 1 a 65535 lux, ideal para controlar la iluminación suplementaria.

Luminosidad
🧪

Sensor pH analógico

Electrodo de pH con módulo de acondicionamiento de señal. Rango 0–14 pH para monitoreo de la solución nutritiva del suelo.

pH del suelo
🌊

Sensor capacitivo de humedad

Sensor de humedad del suelo capacitivo (sin corrosión). Mide el contenido volumétrico de agua en la tierra de cada sector.

Humedad del suelo
💻 Tecnologías · Software

Software y lenguajes

🐍

Python 3

Lenguaje principal del proyecto. Usado para la lógica del servidor, lectura de sensores, automatización y scripting general.

🌐

Flask

Framework web minimalista de Python. Maneja las rutas, la API REST y el renderizado de templates Jinja2.

🎨

HTML / CSS / JS

Frontend del dashboard construido con HTML semántico, CSS personalizado con variables y JavaScript vanilla.

📡

Protocolo I²C / GPIO

Protocolos de comunicación entre la Raspberry Pi y los sensores/actuadores del invernadero.

🗄️ Tecnologías · Datos

Base de datos

🗄️

SQLite

Base de datos relacional embebida, sin servidor. Almacena todas las lecturas de sensores, usuarios, cultivos y configuraciones del sistema.

🔗

SQLAlchemy

ORM de Python para la gestión de la base de datos. Facilita las consultas, migraciones y el modelo de datos del proyecto.

👥 Equipo

Estudiantes de ProA Capital

El proyecto fue desarrollado de forma colaborativa por estudiantes de primero a sexto año. Cada curso aportó desde su nivel de conocimiento y área de trabajo.

Roles por año

1° y 2°
Investigación y ciencias naturales

Estudio de las especies cultivadas, condiciones óptimas de crecimiento, relevamiento de necesidades del invernadero y documentación biológica del proyecto.

3°
Electrónica y hardware

Selección, instalación y calibración de sensores. Cableado del invernadero, montaje de la Raspberry Pi y Arduino, y conexión de actuadores.

4°
Programación backend

Desarrollo del servidor Flask, diseño del modelo de base de datos, lógica de automatización y API REST para la comunicación con el frontend.

5°
Diseño y frontend

Diseño visual del dashboard, implementación de la interfaz web, visualización de datos y experiencia de usuario responsive.

6°
Coordinación y gestión

Dirección técnica del proyecto, integración de todos los módulos, documentación final y presentación ante la comunidad educativa.
